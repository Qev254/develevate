# Develevate Workflows

## Overview
This project is developed using a mobile-first workflow powered by Termux, Git, and lightweight tools.

The goal is to maintain:
- speed
- consistency
- low friction
- full control without reliance on a traditional desktop setup

---

## Development Environment

### Device
- Android (mobile-first)

### Tools
- Termux (CLI environment)
- Git (version control)
- GitHub CLI (`gh`)
- Nano (editing)
- ACode / other editors (optional)
